# 🎨 THÈME CLAIR (DEFAULT STRATÉGIQUE)

## Pourquoi le clair ?

Psychologiquement :

Le blanc évoque :

- Clarté
- Organisation
- Transparence
- Rigueur

Pour un étudiant stressé :

> Le blanc rassure.
> 

Un dark par défaut aurait envoyé :

- Tech
- Puissance
- Mais moins accessible

Donc le choix light-first est stratégique pour la conversion.

# 🟣 Pourquoi le violet (subtil) ?

Psychologie :

- Intelligence
- Innovation
- Futur
- Stratégie
- Technologie maîtrisée

Différenciation :

Le marché emploi est saturé en :

- Bleu institutionnel
- Bleu corporate
- Bleu + rose startup RH

Le violet te positionne :

- Plus tech
- Plus stratégique
- Plus distinctif

Mais tu veux qu’il soit subtil.

Donc :

- Pas de fond violet massif
- Pas de dégradés agressifs
- Utilisé pour :
    - Boutons primaires
    - États actifs
    - Indicateurs importants
    - Micro-éléments

Le violet devient un signal d’intelligence.

Vert = progression.

Il doit servir pour :

- Score CV

- Amélioration réussie

Psychologie :

- Réussite
- Croissance
- Validation
- Résultat concret

Il soutient la promesse :

> Tu progresses.
